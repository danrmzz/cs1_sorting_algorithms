# CS1 Sorting Algorithms


| Algorithm       | Complexity          | Notes                                  |
|-----------------|---------------------|----------------------------------------|
| Selection Sort  | O(n²)               |                                        |
| Bubble Sort     | O(n²)               |                                        |
| Insertion Sort  | O(n²)               |                                        |
| Merge Sort      | O(n log n)          |                                        |
| Quick Sort      | Pivot Dependent     | Average: O(n log n), Worst: O(n²)      |
